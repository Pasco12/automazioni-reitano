10. SICUREZZA
-------------
Sono state aggiunte/mantenute le seguenti protezioni:
- Password con hash PBKDF2.
- Token firmati HMAC con scadenza.
- 2FA TOTP per admin.
- Rate limit su login e recupero password.
- Header HTTP di sicurezza.
- Validazione token Google lato backend.
- Upload filtrato per tipo file.
- API no-cache.
- Nessun link admin nel sito pubblico.
- Nessun dato bancario nel sito pubblico.
- Admin non salvato in localStorage.
- File security.json escluso dal pacchetto finale.

Nota: nessun sito può essere considerato impenetrabile al 100%, ma il progetto è stato rinforzato e testato per una pubblicazione iniziale sicura.
